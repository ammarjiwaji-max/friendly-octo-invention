# GitHub Actions Secrets Setup Guide\n\n## Setup Instructions\n\n### 1. Generate Gmail App Password\n\n1. Go to: https://myaccount.google.com/apppasswords\n2. Select \"Mail\" and \"Windows Computer\"\n3. Google will generate a 16-character password\n4. Copy this password\n\n### 2. Add Repository Secrets\n\n1. Go to: https://github.com/ammarjiwaji-max/friendly-octo-invention/settings/secrets/actions\n2. Click \"New repository secret\"\n3. Add these secrets:\n\n| Secret Name | Value |\n|---|---|\n| EMAIL_SENDER | ammar.jiwaji@gmail.com |\n| EMAIL_PASSWORD | Your 16-char Gmail App Password |\n| EMAIL_RECIPIENT | ammar.jiwaji@gmail.com |\n| BOT_SCHEDULE_TIME | 06:00 |\n| TIMEZONE | Africa/Dar_es_Salaam |\n\n### 3. Test the Workflow\n\n1. Go to: https://github.com/ammarjiwaji-max/friendly-octo-invention/actions\n2. Select \"Manual News Fetch\" workflow\n3. Click \"Run workflow\"\n4. Check logs and verify email was sent\n\n### 4. Verify Daily Schedule\n\nThe \"Daily Financial News Bot\" workflow will automatically run every day at 6 AM UTC.\n\nTo verify it's scheduled:\n1. Go to Actions tab\n2. Click \"Daily Financial News Bot\"\n3. You should see the cron schedule: `0 6 * * *` (6 AM UTC daily)\n\n## Timezone Note\n\nThe workflow uses UTC time (cron: `0 6 * * *`). If you want a different time:\n\n- **6 AM UTC** = 9 AM EAT (East Africa Time)\n- To change, edit `.github/workflows/daily-news.yml` and modify the cron expression\n- Use: https://crontab.guru/ to calculate your desired time\n\nFor Tanzania (EAT, UTC+3):\n- 6 AM UTC = 9 AM EAT\n- To send at 6 AM EAT, use cron: `0 3 * * *`\n\n## Troubleshooting\n\n**Email not sending?**\n- Verify Gmail App Password is correct\n- Check 2-Step Verification is enabled\n- Ensure secret values have no extra spaces\n- Check workflow logs for error messages\n\n**Workflow not running?**\n- Verify secrets are set correctly\n- Check GitHub Actions is enabled for the repository\n- Review workflow logs: Actions tab → Workflow → Run\n\n## Testing Locally\n\nBefore deploying, test the email sender:\n\n```bash\n# Set environment variables\nexport EMAIL_SENDER=\"ammar.jiwaji@gmail.com\"\nexport EMAIL_PASSWORD=\"your-app-password\"\nexport EMAIL_RECIPIENT=\"ammar.jiwaji@gmail.com\"\n\n# Run test script\npython test_email.py\n```\n\nExpected output:\n```\n✅ SUCCESS! Email sent successfully!\n   📬 Email delivered to: ammar.jiwaji@gmail.com\n   📅 Timestamp: 2026-07-15 14:30:00 UTC\n```\n"
